# دليل المساهمة في Noorify Bot
شكراً لاهتمامك بالمساهمة في Noorify! نرحب بجميع المساهمات من المطورين والمصممين والكتاب.

---

## 📋 قواعد السلوك

نحن ملتزمون بتوفير بيئة ترحيبية للجميع. يرجى:

✅ التعامل باحترام مع الجميع  
✅ قبول النقد البناء  
✅ التركيز على ما هو الأفضل للمجتمع  
✅ إظهار التعاطف مع الآخرين  

---

## 🚀 خطوات المساهمة

### 1. Fork المشروع
```bash
# انسخ المستودع لحسابك
```

### 2. استنسخ الفرع الخاص بك
```bash
git clone https://github.com/your-username/noorify-azkar-bot.git
cd noorify-azkar-bot
```

### 3. أنشئ فرع جديد
```bash
# للميزات الجديدة
git checkout -b feature/your-feature-name

# لإصلاح الأخطاء
git checkout -b fix/your-bug-fix
```

### 4. اكتب الكود
```bash
# تأكد من:
- اتباع نمط الكود الحالي
- إضافة تعليقات واضحة
- اختبار الكود
```

### 5. Commit التغييرات
```bash
git add .
git commit -m "Add: description of changes"
# أو
git commit -m "Fix: description of fixes"
```

### 6. Push للفرع
```bash
git push origin feature/your-feature-name
```

### 7. افتح Pull Request
- اشرح الميزة بوضوح
- أضف أي تفاصيل مهمة
- اربط أي Issues ذات صلة

## 📝 معايير الكود

### نمط الكود:
```javascript
// استخدم const و let، لا تستخدم var
const myVar = "value";

// استخدم أسماء واضحة
const userPreferences = {};

// أضف تعليقات للأجزاء المعقدة
// تعليق يشرح الكود
const result = complexFunction();

// استخدم async/await بدل .then()
async function getData() {
    try {
        const data = await fetchData();
        return data;
    } catch (err) {
        console.error("Error:", err);
    }
}
```

### معايير أخرى:
- ✅ استخدم العربية في التعليقات
- ✅ استخدم الإنجليزية في أسماء المتغيرات
- ✅ اكتب كود نظيف وسهل الفهم
- ✅ تجنب الكود المعقد غير الضروري

## 🐛 الإبلاغ عن الأخطاء

عند الإبلاغ عن خطأ:

1. **استخدم عنوان واضح:**
   ```
   [BUG] الأذكار لا تصل في التذكيرات
   ```

2. **اشرح الخطأ:**
   ```
   الخطوات:
   1. فعل التذكيرات
   2. انتظر الساعة
   3. لا تأتي الرسالة
   ```

3. **أضف الفحوصات:**
   ```
   البيئة:
   - Node.js v16
   - Railway Platform
   ```

---

## ✨ اقتراح ميزات جديدة

عند اقتراح ميزة:

1. **استخدم عنوان واضح:**
   ```
   [FEATURE] إضافة تذكيرات صوتية
   ```

2. **اشرح الفائدة:**
   ```
   هذه الميزة ستساعد المستخدمين الذين...
   ```

3. **أضف أمثلة:**
   ```
   الاستخدام:
   /sound_on - تفعيل الصوت
   /sound_off - إيقاف الصوت
   ```

## 📚 أفكار للمساهمة

### ميزات يمكنك العمل عليها:
-  إضافة لغات جديدة (الإنجليزية، الفرنسية)
-  تصنيفات للأذكار (الصلاة، الطعام، السفر)
-  تذكيرات صوتية
-  واجهة ويب لإدارة البوت
-  نظام تقييم الأذكار
-  مشاركة الأذكار مع الأصدقاء

### إصلاحات يمكنك العمل عليها:
-  تحسين سرعة الاستجابة
-  إضافة معالجة أخطاء أفضل
-  تحسين التوثيق
-  إصلاح الأخطاء المعروفة
