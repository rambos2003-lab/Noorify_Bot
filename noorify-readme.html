الأمر الوصف الاستخدام
/start الترحيب والقائمة الرئيسية ابدأ هنا أولاً
/help عرض جميع الأوامر والمساعدة إذا احتجت مساعدة

البدء السريع — للمطورين
bash — installation
# 1. Clone the repository git clone https://github.com/rambos2003-lab/noorify-azkar-bot cd noorify-azkar-bot # 2. Install dependencies npm install # 3. Setup environment echo "TELEGRAM_BOT_TOKEN=your_token_here" > .env # 4. Start the bot npm start
1

متطلبات النظام

تأكد من وجود Node.js v14+ و npm على جهازك، وأنشئ بوت Telegram عبر @BotFather

2

استنسخ المشروع وثبّت المكتبات

نفّذ أوامر git clone و npm install لتجهيز البيئة

3

أضف متغيرات البيئة

أنشئ ملف .env وضع فيه الـ TELEGRAM_BOT_TOKEN

4

شغّل البوت 🎉

نفّذ npm start وانتقل إلى البوت على Telegram
