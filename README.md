# Noorify Bot ۩

بوت ذكي يساعدك على الاستمرار في ذكر الله يومياً بشكل تلقائي وتفاعلي

إذا أعجبك المشروع لا تنسَ دعمه بنجمة ⭐ 

## جرّب البوت مباشرة: https://t.me/Noorify_bot

## المميزات

- إرسال أذكار تلقائي (كل ساعة / يومي)
- تخصيص أوقات التذكير
- ردود ذكية على الأذكار
- تصميم بسيط وسهل الاستخدام
- تنبيهات تساعد على الاستمرارية

---

## لماذا هذا المشروع؟

معظم تطبيقات الأذكار تقليدية ومملة  
هذا البوت يهدف إلى:

- جعل الذكر عادة يومية مستمرة
- تقديم تجربة تفاعلية
- تسهيل الوصول للأذكار بدون تعقيد
