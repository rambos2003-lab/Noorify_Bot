 
 # Noorify Bot 🤖

بوت تيليجرام تفاعلي يهدف إلى مساعدة المستخدمين على المداومة على ذكر الله، وتوفير مكتبة رقمية إسلامية.

## المميزات 💎 
- **أذكار دورية:** إرسال ذكر كل ساعة للمشتركين.
- **مكتبة رقمية:** إمكانية تحميل كتب إسلامية بصيغة PDF.
- **تفاعل ذكي:** الرد على عبارات التسبيح والتحميد.
- **سهل الإدارة:** أوامر بسيطة لتفعيل وإيقاف التذكيرات.

## التقنيات المستخدمة ⚙️
- [Node.js](https://nodejs.org/)
- [node-telegram-bot-api](https://github.com/yagop/node-telegram-bot-api)
- [dotenv](https://github.com/motdotla/dotenv)
