## 📖 نظرة عامة
**Noorify** هو بوت متطور صمم بعناية ليكون رفيقك الروحي، يساعدك على مداومة ذكر الله يومياً بشكل تلقائي وتفاعلي.

## ✨ المميزات
* 📱 **أذكار ذكية:** تنوع وموثوقية في أكثر من 25 ذكر.
* ⏰ **تذكيرات مخصصة:** 4 فترات زمنية تتكيف مع جدولك (30د، 1س، 3س، 6س).
* 📚 **مكتبة رقمية:** وصول سريع لأذكار الصباح، المساء، وحصن المسلم.
* 📊 **تتبع دقيق:** إحصائيات يومية وعداد للأوامر الأكثر استخداماً.
* 🎮 **تفاعل ذكي:** ردود سريعة وواجهة سهلة الاستخدام.
* 🔒 **خصوصية عالية:** جميع البيانات محفوظة محلياً.

## 🚀 التثبيت
```bash
# 1. استنسخ المستودع
git clone [https://github.com/rambos2003-lab/Noorify_Bot.git](https://github.com/rambos2003-lab/Noorify_Bot.git)
cd Noorify_Bot

# 2. ثبت الاعتمادات
npm install

# 3. اضبط متغيرات البيئة
echo "TELEGRAM_BOT_TOKEN=your_token_here" > .env

# 4. شغل البوت
npm start
