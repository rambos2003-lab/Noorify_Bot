#  Noorify Bot - بوت الأذكار الذكي

## مقدمة
**Noorify** هو بوت Telegram ذكي يساعدك على الاستمرار في ذكر الله يومياً بشكل تلقائي وتفاعلي. صمم بعناية ليكون رفيقك الروحي في رحلتك نحو مداومة الذكر.

### 🎯 جرّب البوت الآن
**[@Noorify_bot](https://t.me/Noorify_bot)** - ابدأ رحلتك الآن!

## 🚀 البدء السريع

### للمستخدمين (بدون تثبيت)
1. افتح Telegram
2. ابحث عن [@Noorify_bot](https://t.me/Noorify_bot)
3. اكتب `/start`
4. ابدأ رحلتك! 🌟

### للمطورين (تثبيت محلي)

#### المتطلبات:
- Node.js v14 أو أحدث
- npm أو yarn
- Telegram Bot Token

#### خطوات التثبيت:
```bash
# 1. استنسخ المشروع
git clone https://github.com/username/noorify-azkar-bot
cd noorify-azkar-bot

# 2. ثبت المكتبات
npm install

# 3. أنشئ ملف .env
echo "TELEGRAM_BOT_TOKEN=your_token_here" > .env

# 4. شغل البوت
npm start
```
