## 🌟 نبذة عن المشروع

**Noorify Bot** هو بوت تيليجرام ذكي مخصص للأذكار والقرآن الكريم والتسبيح،  
مصمم ليكون خفيف، سريع، وقابل للتطوير بشكل احترافي.

يهدف إلى:
- 📿 تعزيز الذكر اليومي
- 📖 نشر الأذكار والقرآن
- 🧠 تجربة مستخدم سلسة
- ⚡ أداء عالي واستقرار كامل

---

## 🚀 المميزات

- 📿 مسبحة ذكية (Tasbeeh System)
- 🌙 أذكار الصباح والمساء والنوم
- 📚 مكتبة إسلامية (PDF System)
- 📊 إحصائيات المستخدمين
- ⚙️ إعدادات متقدمة
- 🔔 نظام تذكيرات تلقائي
- 💬 دعم المجموعات
- 🧩 أزرار تفاعلية Inline Keyboard

---

## 🧱 التقنيات المستخدمة

- Node.js / TypeScript
- Telegram Bot API
- Modular Architecture
- Clean Code Principles
- Scheduler System
- JSON / Storage Layer
